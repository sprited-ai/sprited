sprited

8-direction game character sprites from one prompt — template-guided image models, local BiRefNet-ToonOut matting, model self-review. Open source.

github npm model (HF)

one command

GEMINI_API_KEY=... npx sprited gen char -d "a small forest fairy with green wings"

→ spritesheet (8 directions) + animated turnaround + entity.json + a yaml that reproduces the exact build. Generation by Nano Banana Pro; background removal by BiRefNet-ToonOut running locally via onnxruntime; the result is reviewed and repaired by the image model itself.

made with sprited

npx sprited build lisa -d "Character is Lisa and she is a fairy."
npx sprited build monet -r ./monet.reference.png
npx sprited build blue-dress -r ./blue-dress.reference.png

how it works

An example-anchored template (worked example on top, your reference + empty slots below) goes to the image model, which completes the pattern — no fine-tuning, no LoRA. Five views are generated (S SE E NE N), three are mirrored, each is matted by ToonOut, and the assembled turnaround goes back to the model as a labeled 3×3 grid: "any errors? fix them and report the changes."
